Detailed Description

FreeRTOS Resources Configuration for STM32H753ZI Stepper Motor Control.

This file declares all FreeRTOS task handles, queues, semaphores, and mutexes used throughout the system. These are the shared RTOS resources that enable inter-task communication and synchronization.
