Detailed Description

SSOT configuration for motor efficiency optimization system (FTR-009)

This file provides the Single Source of Truth (SSOT) for all optimization-related configurations including algorithm parameters, performance targets, safety limits, and integration settings.

CRITICAL: This is the SSOT for optimization configuration. All optimization modules must reference these values. Never hardcode optimization parameters elsewhere.
