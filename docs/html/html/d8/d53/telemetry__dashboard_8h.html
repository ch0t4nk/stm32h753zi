Detailed Description

Real-Time Telemetry Dashboard and HTTP JSON API.

Author
STM32H753ZI Project Team
Date
August 2025
Version
1.0

This module provides a lightweight HTTP server with JSON API for real-time motor control telemetry, performance metrics, and system diagnostics over STLink USB CDC interface with industrial-grade monitoring capabilities.
