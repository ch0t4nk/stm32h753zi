Detailed Description

Mock STM32H7xx HAL header for unit testing.

This file provides a comprehensive mock of the STM32H7xx HAL system to enable compilation of STM32-specific driver code in unit testing environments without requiring the full STM32 HAL library.

Note
This is a MOCK header for TESTING ONLY
