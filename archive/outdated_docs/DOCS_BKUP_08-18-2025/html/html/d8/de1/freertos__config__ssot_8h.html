Detailed Description

FreeRTOS Single Source of Truth Configuration for STM32H753ZI Stepper Motor Control.

This file defines all FreeRTOS configuration parameters in a centralized location following the project's SSOT principles. All RTOS-related code must reference these definitions rather than hardcoding values.
